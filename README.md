## Overview

Cabinet Designer combines cabinet planning, part layout, and fabrication prep in one browser-based interface. It is built for woodworkers who need a practical way to shape assemblies, inspect proportions in 3D, and move from early concept to shop-ready output without jumping between separate applications.

This project fits both professional shops and hobby workflows where accurate parts, clear material handling, and exportable deliverables are important. Its parametric workflow makes iterative changes straightforward, and the preview plus reporting tools let you check the design before sending it to production or sharing it with collaborators.

---

## What It Does

- Create cabinet assemblies in 2D and 3D
- Inspect models with a live 3D preview
- Use a material and hardware library for common cabinet components
- Produce cut lists automatically from the active design
- Run panel and nesting optimization for improved layout use
- Export Excel and PDF files for planning and documentation
- Generate CNC router toolpaths and G-code for fabrication workflows
- Export SCM Maestro-compatible files for downstream production use

## Using the App

1. Start a new cabinet project or open one you already have.
2. Set the cabinet dimensions, panels, and hardware.
3. Review the assembly in 2D, then switch to the 3D preview to inspect the model.
4. Generate the cut list and export it to Excel or PDF.
5. Use the nesting tools to evaluate layout efficiency.
6. Export CNC-related output, including toolpaths, G-code, STEP, or SCM Maestro formats when needed.

Typical workflow:

- Sketch the cabinet structure
- Refine the parametric values
- Check the rendered result
- Export documentation and machine-ready data
